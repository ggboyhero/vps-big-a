
## 安装脚本

- 支持快捷方式启动，安装完毕后，shell输入【**vasma**】即可打开脚本，脚本执行路径[**/etc/v2ray-agent/install.sh**]

- Latest Version【推荐】

```
wget -P /root -N --no-check-certificate "https://raw.githubusercontent.com/ggboyhero/vps-big-a/master/install.sh" && chmod 700 /root/install.sh && /root/install.sh
```

- shadowsocks 动态IP白名单模式【Beta】

```
wget -P /root -N --no-check-certificate "https://raw.githubusercontent.com/ggboyhero/vps-big-a/dev_ss/install.sh" && chmod 700 /root/install.sh && /root/install.sh
```
